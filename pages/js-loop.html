Циклы

В программах бывает много повторяющихся одинаковых действий. Самый очевидный способ реализовать повторяющуюся функциональность — повторять строки кода:
			

				Скопировать код
				JAVASCRIPT
			

			
				console.log('Никаких больше вечеринок');
				console.log('Никаких больше вечеринок');
				console.log('Никаких больше вечеринок');
				console.log('Никаких больше вечеринок');
				console.log('Никаких больше вечеринок');
Было бы классно не копировать один и тот же код, а сказать: «Исполни команду console.log 5 раз». Хорошие новости: это возможно, в JavaScript есть цикл for.
Цикл позволяет повторять похожие действия, задав условия повторения. Конструкция цикла похожа на конструкцию условия if. Сначала идёт ключевое слово for, потом круглые скобки, а затем фигурные. В фигурных скобках размещают код, который нужно повторять:
			

				Скопировать код
				JAVASCRIPT
			

			for () {
  console.log('Никаких больше вечеринок');
}
А в круглых — три условия цикла, разделённых точкой с запятой. Во-первых, повторения нужно считать. Оттого первое условие — это начальное значение переменной-счётчика:
Скопировать кодJAVASCRIPT
for (let i = 0;) {
    console.log('Никаких больше вечеринок');
}
Переменная-счётчик отвечает за то, сколько раз будет выполняться код в фигурных скобках. Чтобы ограничить это количество, есть второе условие, оно задаёт конечное значение переменной:
Скопировать кодJAVASCRIPT
for (let i = 0; i <= 4;) {
  console.log('Никаких больше вечеринок');
}
// повторяем текст о вечеринках не более 5 раз, считая от 0 до 4
Но от начального значения к конечному нужно как-то идти. Третье условие говорит о том, с каким шагом считать. Например, увеличивать счётчик на единицу:
Скопировать кодJAVASCRIPT
for (let i = 0; i <= 4; i = i + 1) {
  console.log('Никаких больше вечеринок');
}
Такой цикл выведет в консоль фразу «Никаких больше вечеринок» 5 раз.
Массивы хорошо подходят, чтобы перебирать их элементы в цикле. Такой код выведет в консоль все элементы массива array, тем самым сосчитав до пяти:
Скопировать кодJAVASCRIPT
let array = ['животные', 'растения', 'грибы', 'микроорганизмы', 'вирусы'];

for (let i = 0; i <= 4; i = i + 1) {
  console.log(array[i]);
}
// животные
// растения
// грибы
// микроорганизмы
// вирусы
Переменная i, изменяясь в цикле, играет роль индекса, по которому можно поочерёдно достать каждый элемент.
